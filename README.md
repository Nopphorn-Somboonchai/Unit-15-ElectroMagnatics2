# Unit-15-ElectroMagnatics2

🛠️ Development Guidelines (แนวทางการพัฒนาแอปพลิเคชัน)
📐 Mathematics Rendering
สมการทางคณิตศาสตร์และสูตรฟิสิกส์ทั้งหมดต้องแสดงผลอย่างสวยงามถูกต้องตามมาตรฐานทางวิชาการ โดยใช้เครื่องมือแสดงผลอย่าง MathJax หรือ KaTeX (หรือการประมวลผล LaTeX อื่น ๆ)

เงื่อนไขของโหมดสอบเก็บคะแนน : มีตรรกะการนำค่า "เลขที่ของนักเรียน" เข้าไปในโจทย์ปัญหา เพื่อให้ได้โจทย์ที่ตัวเลขไม่ซ้ำกัน แม้ว่านักเรียนได้โจทย์ข้อเดียวกัน ก็สามารถแยกคำตอบได้ 

💻 Technology Stack
พัฒนาโดยใช้ Pure Client-Side Technologies:
HTML5 (โครงสร้างหน้าเว็บ)
Vanilla CSS3 (การจัดวางสไตล์ แอนิเมชัน และ Responsive Layout)
Vanilla JavaScript (ระบบกิจกรรมการเรียนรู้แบบโต้ตอบ ตัวจำลองสถานการณ์ และ Quiz)

📱 Responsive Design
Mobile-First Approach: ออกแบบเพื่อรองรับอุปกรณ์พกพาเป็นหลัก ก่อนปรับขยายให้พอดีกับ Tablet และ Desktop

🎨 UI/UX & Visual Requirements
Modern Educational Web App: สวยงาม ทันสมัย ดึงดูดสายตานักเรียน
Card-Based Layout: จัดแบ่งข้อมูลออกเป็นสัดส่วนด้วยการใช้การ์ด เพื่อง่ายต่อการสแกนอ่าน
Theme & Colors: ใช้โทนสีที่สอดคล้องกับหัวข้อแม่เหล็กไฟฟ้า (เช่น แดง น้ำเงิน)
Rich Visuals: หลีกเลี่ยงหน้าตัวอักษรยาวเหยียด เน้นใช้ SVG, แผนภาพกระชับ, และ CSS Animation เฉพาะที่ส่งเสริมการเรียนรู้
Interactive Controls: รองรับทั้ง Touch Screen และ Mouse

♿ Accessibility
รองรับการควบคุมด้วยคีย์บอร์ด (Keyboard Navigation)
ใส่ aria-label ให้กับปุ่มและสื่อปฏิสัมพันธ์ที่ไม่มีป้ายกำกับข้อความชัดเจน
รักษาระดับ Contrast ของสีตัวอักษรกับพื้นหลังให้อ่านง่าย

⚡ Performance
เน้นเขียนด้วย Vanilla JavaScript
ลดการพึ่งพา External Library หรือ Asset ภายนอกให้มากที่สุด
หลีกเลี่ยงการใช้ไฟล์ภาพบิตแมปขนาดใหญ่ที่ไม่จำเป็น ให้เน้นใช้ SVG หรือ CSS แทน

📚 Content Requirements
เนื้อหาวิชาการระดับเดียวกับหนังสือ PhysicsBook5
ใช้ภาษาไทยเป็นหลัก โดยมีศัพท์ภาษาอังกฤษกำกับคำเฉพาะทางฟิสิกส์อย่างเหมาะสม
อธิบายเนื้อหาสั้น กระชับ เข้าใจง่าย และมีความถูกต้องแม่นยำทางฟิสิกส์
พยายามสอดแทรกตัวอย่างและกรณีศึกษาจากสถานการณ์จริงในชีวิตประจำวันเพื่อความเข้าใจ
